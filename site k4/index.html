Het schrift

3500 vC.

Lees meer

Rond 3500 vC. Ontstond het schrift door de toenemende handel waarbij de voorraden en/of regels genoteerd moesten worden. Vermoedelijk is het schrift ontstaan in Soemer en later over de rest van de wereld verspreid. De Chinese taal is waarschijnlijk afhankelijk ontwikkeld). Schrijven is een manier waarop je taal kan worden gebruikt. Taalgebruik is gebaseerd op geschreven of gesproken taal. Bij schrijven gaat het om het schrijven van woorden en zinnen. Het schrift is een manier om taal grafisch weer te geven waarmee boodschappen konden worden doorgegeven. Het schrift heeft een ontwikkeling meegemaakt van pictografisch naar het alfabetisch schrift.

Beeldschermen

1980

Lees meer

Een beeldscherm is een apparaat waarop beelden kunnen worden getoond. In 1980 ontstonden de eerste beeldschermen. In diezelfde tijd kwamen beeldschermen veel voor in de vorm van computers. Deze schermen waren monochrome beeldschermen zonder kleur. In de afgelopen 20 jaar zijn de beeldschermen ontwikkeld naar beeldschermen met een groot kleurenpaneel en een hoge resolutie. Tegenwoordig maken veel beeldschermen gebruik van lcd.

Het world wide web

1989

Lees meer

Het world wide web, ook wel bekend als het web, is een verzameling van documenten en toepassingen die wereldwijd worden gedeeld. Al sinds 1989 is Tim Berners-Lee bezig met het ontwikkelen van het concept van het web. Op 6 augustus 1991 plaatste hij een overzicht van het concept in een nieuwsgroep. Hiermee werd het project publiekelijk toegankelijk. Het web bevat miljarden pagina’s die zijn opgebouwd in website die worden weergegeven op een webserver. Op de webserver kunnen onder andere HTML-pagina’s worden weergegeven maar ook andere pagina’s met afbeeldingen, muziek en video’s.

Ontstaan HTML

1991

Lees meer

HTML is een afkorting voor HyperText Markup Language. HTML is een opmaaktaal voor documenten die gebruikt kunnen worden op het world wide web. De HTML-code kan door de browser gelezen worden waarna het als een pagina wordt weergegeven. In 1991 werd HTML publiekelijk gemaakt door Tim Berners-Lee. HTML bevat Semantische tekst die de inhoud van een webpagina weergeeft. Met HTML krijg je geen opmaak, het is puur tekstueel.

Ontstaan CSS

1996

Lees meer

Om een HTML-pagina opmaak te geven gebruik je CSS. CSS is een afkorting voor Cascading Style Sheets, ook wel stijlbladen genoemd. Sinds 1994 werd aan het concept gewerkt Hakon Wium Lie. In 1995 werd het concept voorgesteld op het ‘Mosaic and the Web’ conferentie in Chicago. In 1996 werd CSS publiekelijk gepubliceerd. Met CSS kan de vormgeving worden bepaald van elementen op de webpagina.

Responsive webdesign

2010

Lees meer

In 2010 is het begrip responsive design tot leven gekomen. Responsive design houdt in dat je een webpagina zo maakt dat hij op elke schermgrootte zijn eigen layout heeft. Op deze manier is de website voor elke context geoptimaliseerd en is hij op elk scherm het beste te lezen.
